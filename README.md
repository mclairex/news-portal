# Ponto de Vista

Portal de notícias e análises sobre Política, Esportes e Tecnologia.

## Descrição
O "Ponto de Vista" é um portal web responsivo, desenvolvido em HTML, CSS e JavaScript, com foco em notícias atuais e análises de temas relevantes do Brasil e do mundo. O projeto possui páginas dedicadas para cada editoria, layout moderno e recursos de interação para o usuário.

## Funcionalidades
- Página inicial com manchete principal e destaques das editorias
- Seções separadas para Política, Esportes e Tecnologia
- Layout responsivo e estilizado
- Área de feedback com avaliação por estrelas
- Links para redes sociais e contato

## Estrutura do Projeto
```
news-portal/
├── index.html              # Página principal
├── politica.html           # Página de política
├── esportes.html           # Página de esportes
├── tecnologia.html         # Página de tecnologia
├── css/
│   └── styles.css          # Estilos customizados
├── js/
│   └── script.js           # Script de interação (feedback)
├── assets/                 # Imagens de notícias
├── imagens/                # Ícones e logos de redes sociais
└── ...
```

## Como Executar
1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Navegue pelas seções usando o menu ou os links das páginas.

Não é necessário backend ou instalação de dependências.
